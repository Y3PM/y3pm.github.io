如何给品牌起一个好名字？

2021-01-08

近期要给公司的新产品注册一个品牌，于是由产品部发起了一个“品牌命名征集”，经过一轮征集和投票后，发现几乎都不可用。任务又重新回到了产品部身上……

虽然有过几次给品牌命名的经验，主要是基于所属行业、行业历史、相关竞品、品牌定位等。却也一直没有个完整的方法和标准。趁着这次机会，总结一下。

定位是自身产品和竞品进行深入分析，对消费者的需求进行准确判断，确定产品与众不同的优势及在消费者心中的独特地位，并传达给目标的动态过程。--《定位》艾·里斯

品牌命名的过程就是一个定位的过程，好的品牌名可以有助于占领用户心智，谁先占据用户心智的有利位置，谁就掌握了核心竞争力。

定位

如何找到自己最精准的定位

需要研究三个方面，即作为定位基础的用户认知、定位导向的竞争对手和定位条件的自身。

  • 用户研究需要进行两个方面的思考：界定主要目标群体 和 研究用户是怎么思考的。
  • 研究竞争对手进行两个方面的思考：界定主要竞争对手 和 发掘顾客为什么购买对手的理由。
  • 自身条件分析进行两个方面的思考：外部资源（顾客已有认知） 和 内部资源（资金、能力）。

界定主要竞争对手的三条标准：

  1. 市场比自己大的
  2. 对方的产品和自己是替代关系的
  3. 在替代过程中，能凸显自身的优势

定位的三个方法：

  1. 抢先占位。在用户的心智中找到一个空白位置，动用一切能调动的资源，去成为这个空白位置的第一名。或占据某个类别或者领域的第一，成为品类的代表。或抢先占据某个品类内有价值属性的位置。
  2. 关联定位。与第一品牌巧妙关联，建立某种联系，使别人想到第一品牌，马上会想到你。比如直接站在第一品牌的旁边、站在第一品牌的对立面，与其唱反调、组建「俱乐部」，把自己和一些强大品牌的放在一起，让别人产生「你和大品牌属于一个圈子」的认知。
  3. 为竞争对手重新定位。挖掘竞争对手的弱点，针对竞争对手的弱点，给自己进行定位

品牌命名

名不正则言不顺，言不顺则事不成。

品牌命名的重要性

  • 品牌命名是我们能做的最重要的一个营销决定。
  • 用户是根据认知，尤其是根据对品牌名字的感受和理解，来选择买还是不买。

品牌命名的技巧

  • 启动定位
  • 容易记忆

品牌命名的基本的原则

  1. 第一个原则是品牌命名要有「独特性」。
  2. 第二个原则是要重视发音，忌用多音字、生僻字、谐音。利用品牌发音和大脑中已经存在的声音产生关联、形成一种节奏感、尽量使用开口音。

品牌命名的方法和流程

一句话概括：先有定位，后有命名

  1. 第一步：明确定位。一个有价值的品牌名，一定是一个产品品类或者产品特性的代表。品牌命名的任务，就是让大家更加确定，这个品牌在它代表的品类中占据着独特的地位。
  2. 第二步：抓住关键人群。关键人群是指那些消费量比较高的人群，也叫重度消费人群。他们对普通人的购买决策会产生较大的影响力。
  3. 第三步：收集行业的关键字眼。收集行业关键字眼的作用是，做到不雷同、不接近，不让别人产生混淆的感觉。
  4. 第四步：头脑风暴。这个阶段是先求数量，在有了一定数量后，再在里面挑选更吻合定位、更简洁、更有差异性的命名。
  5. 最后一步；命名测试。通过命名测试，可以判断出市场对命名的接受程度和消费者的真实想法。

定位口号

凡是深入人心的口号必有「心智附着力」，定位口号使用的一条原则是「能不变，就不变」，重复、重复、再重复。
定位口号需具备的通用要求：尽量包含品牌名、尽量短小精悍，如“百度更懂中文”。

什么样的定位口号最能实现病毒式的传播？

  1. 「病句」式定位口号，突破传统语法的规范，如蒙牛草原奶的“喝蒙牛，就是喝草原”，脑白金的“今年过年不收礼，收礼只收脑白金”；
  2. 「冲突」的定位口号，如“充电5分钟，通话2小时”；
  3. 「诗化」定位口号，如“钻石恒久远，一颗永流传”，“不在乎天长地久，只在乎曾经拥有”；
  4. 「俗语」定位口号，如“农夫山泉有点甜”，“困了，累了，喝红牛”；

有效的定位口号，拥有的最直接、最质朴的功能就是促使人们产生行动。

品类与品牌

用品类来思考，用品牌来表达。

品类，是顾客心智对产品或者服务的存储类别，是消费选择的出发点，是隐藏在品牌背后的关键营销力量。品类，是顾客消费的驱动力。

打造品牌的关键在于开创新品类

新品类，来自老品类的分化，分化是商业界发展的原动力。开创新品类可以使用两种方法：

  1. 技术创新开创「市场中有，心智中无」的新品类；
  2. 利用新趋势，聚焦开创新品类；

在寻找新品类机会的时候，有四个要点需要注意

  1. 「现有市场为零」最佳
  2. 新概念不都是新品类
  3. 避免过度分化
  4. 警惕新品类杀手

身为品类领导者，品牌必须具有从整个品类思考的远见，对品类前景充满信心。
